# Run Earthquake

This section provides instructions on how to run Earthquake and introduces some
notable command-line options available to customize its operation.

```admonish
Content in this section is currently under development. Please check back later for updates.
```

**To start Earthquake after installation:**

```bash
earthquake run
```
